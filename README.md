# Conversor-de-idade-em-dias
Meu primeiro código em C++.

1. O que ele faz?
   - O código solicita quantos anos você tem e após isso realiza um cálculo para converter o valor em "anos" para "dias".
   - idadeDias = idade * 365;
  
2. Como foi construído?
   - Principalmente com os conhecimentos básicos em C++ que adquiri em meus estudos em ADS.
   - Foi feito para eu treinar tanto a linguagem C++, como aprender sobre: Desvio (if e else), condicionais, variáveis e operações com variáveis.
  
O código em seu próposito foi feito com a intenção de praticar e construir conhecimento por meio da prática, não possui aplicabilidade real.
